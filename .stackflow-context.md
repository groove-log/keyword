## Module Definitions

### 1. `worker.py`

Handles asynchronous tasks using Celery, database interaction via `psycopg`, keyphrase extraction, and embedding-based semantic classification.

#### Class: `KeywordExtractor`
* **Purpose**: Torches-free keyphrase extraction using raw embeddings via HTTP.
* **Methods**:
  * `__init__(self, api_url: str, api_key: str, mode: str = "v1")` -> None
  * `_get_embedding(self, text: str) -> list[float]`
  * `_extract_candidates(self, text: str, ngram_range: tuple[int, int]) -> list[str]`
  * `_cosine_similarity(a: list[float], b: list[float]) -> float`
  * `extract_keywords(self, text: str, keyphrase_ngram_range: tuple[int, int] = (1, 2), top_n: int = 3) -> list[tuple[str, float]]`

#### Class: `SemanticClassifier`
* **Purpose**: Classify context or individual phrases using Cosine-Similarity against natural language risk guidelines.
* **Methods**:
  * `__init__(self, extractor, guidelines: dict[str, str], threshold: float = 0.60)` -> None
  * `_ensure_cached(self) -> None`
  * `classify(self, text: str) -> tuple[str, float]`

#### Celery App & Task
* `app`: Celery application instance configured with Redis broker.
* `@app.task`
  `process_keyword_extraction(counseling_id: int, text: str) -> str`

#### Helpers
* `get_db_connection()` -> psycopg connection object

---

### 2. `web_app.py`

FastAPI application that serves the front-end user interface and routes incoming sync/async extraction queries.

#### APIs
* `GET /` -> HTMLResponse: Serves `index.html`.
* `POST /api/extract/sync` -> JSON: Live real-time noun keyword extraction and semantic risk classification.
* `POST /api/extract/async` -> JSON: Queues asynchronous background task.
* `GET /api/history` -> JSON: Queries top 10 logs from PostgreSQL.
