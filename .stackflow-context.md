### 2. `web_app.py`

FastAPI web prototype providing synchronous and asynchronous keyword extraction, live log fetching, and re-analysis trigger APIs.

#### Served Routes
* `GET /`: Serves `index.html` dashboard.
* `GET /guide.html` or `/guide`: Serves `guide.html` technical guide.
* `GET /api/settings`: Returns default guidelines, thresholds, and configuration settings.
* `POST /api/extract/sync`: Computes real-time extraction and returns JSON payload synchronously.
* `POST /api/extract/async`: Enqueues extraction in Celery queue and writes pending row to database.
* `GET /api/history`: Fetches the last 10 log records from `counseling_data`.
* `POST /api/reanalyze/{counseling_id}`: Re-runs 2D classification and extraction for a specific ID.
* `POST /api/reanalyze-all`: Batch re-runs classification and extraction for all rows.
